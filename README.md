Cloud-Based University Portal
**1. Overview**
The Cloud-Based University Portal is a web application developed to simplify student module registration and academic management processes. The system enables students to register, browse modules, and manage academic activities through a secure cloud-hosted platform.

**2. Access the deployed application here:**

Link: https://app655-ajcjd4fxcfdndvd7.uksouth-01.azurewebsites.net/ 

The project was developed using Django and deployed using Microsoft Azure services to provide scalability, reliability, and secure data management.

**3. Functionalities Implemented**
  1. User Registration & OTP Confirmation
    * New users must register with their email address.
    * An OTP (One-Time Password) is sent to confirm registration.
  2. Password Reset
    * Users can reset their password securely via email confirmation.
  3. Django REST API Integration – News API
    * The portal consumes and displays real-time university-related news using REST API.
  4. Google Maps Integration
    * Embedded Google Maps feature to display university location.
  5. University Video Resource
    * Integrated YouTube video relevant to university life and portal demonstration.
  6. Website Favicon & Logo
    * A custom favicon and university logo have been added to the site for branding.
  7. Contact form with email confirmation
    * Users can contact the university through a form.
    * Email confirmation is sent automatically upon form submission.
  8. Pagination and Search facility
    * Application implements pagination or search facility

**5. System Architecture**
  1. Users interact with the portal through a web interface.
  2. Django handles backend logic and authentication.
  3. Azure SQL stores application data.
  4. Azure Storage stores media and static files.
  5. External APIs provide dynamic content and integrations.
  6. Azure Web App hosts the deployed application.
